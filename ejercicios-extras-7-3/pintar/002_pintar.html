Let's paint

CAS

Crear un programa que pueda pintar un cuadrado

  • Preguntar al usuario que escriba un número para determinar el lado del cuadrado
  • Para los bordes del cuadrado se utilizará 'chartOne'
  • Para los contenido del cuadrado se utilizará 'chartTwo'

Variables:

  • chartOne = '#'
  • chartTwo = '*'
No se podrá utilizar la instrucción "repeat"
Ejemplo para el numero 5
