Examen: Gestión de Inventario de Biblioteca

Estás desarrollando un programa para gestionar el inventario de una biblioteca. Los datos se almacenan en tres arrays:

  • Un array que contiene los títulos de los libros.
  • Un array que contiene la cantidad de ejemplares disponibles de cada libro.
  • Un array que contiene el precio por ejemplar de cada libro.

Reglas del Ejercicio:

  1. Cada elemento en los arrays corresponde a un libro específico.
  2. Los valores en los arrays pueden incluir "null", "undefined" o valores negativos.
  3. Escribe una función que reciba tres arrays: "titulosLibros", "cantidades", y "precios".
  4. La función debe calcular el valor total del inventario de la biblioteca.
  5. Los valores negativos, "null", y "undefined" no deben afectar el cálculo del valor total.
  6. La función debe aceptar tres arrays de la misma longitud y devolver un mensaje de error si no lo son.
  7. La función debe devolver el valor total del inventario como un número.
  8. Los arrays de entrada no deben ser modificados.

Ejemplo:

Entrada:
const titulosLibros = ["El Quijote", "Cien Años de Soledad", "1984", "Moby Dick", "Hamlet"];
const cantidades = [5, null, 10, undefined, -2];
const precios = [20, 15, 10, 25, 30];
Salida esperada:
El valor total del inventario es: 350

Preguntas del Examen:

  1. Parte 1: Escribe una función llamada `validarLongitudArrays` que reciba tres arrays y verifique si tienen la misma longitud. Devuelve `true` si coinciden o `false` si no.
  2. Parte 2: Escribe una función llamada `filtrarDatosInvalidos` que reciba un array y elimine valores inválidos como `null`, `undefined` o números negativos. Devuelve un nuevo array sin modificar el original.
  3. Parte 3: Escribe una función llamada `calcularValorInventario` que reciba tres arrays (`titulosLibros`, `cantidades`, y `precios`) y calcule el valor total del inventario siguiendo las reglas mencionadas. Devuelve el valor total como un número.
  4. Parte 4: Modifica la función `calcularValorInventario` para que devuelva un mensaje de error si los arrays no tienen la misma longitud o si alguno contiene datos inválidos después del filtrado.
  5. Parte 5: Escribe al menos dos casos adicionales con diferentes datos para probar tus funciones y asegúrate de que funcionan correctamente.
